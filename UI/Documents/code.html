Using Logic Verbs

Control the flow of your bot with conditional logic blocks. Create complex interactions by chaining If/Else statements, loops, and randomized outcomes.

schedule Last updated: Oct 24, 2023
timer 5 min read

call_split The "If/Else" Verb

The If/Else verb is the fundamental building block of logic in BotPanel. It allows your bot to make decisions based on variables, user roles, channel types, or random chance.

Visual Editor Preview

Configuration Properties

When you drag an If/Else verb onto the canvas, you'll need to configure the condition. BotPanel supports standard comparison operators (==, !=, >, <).

Property Type Description
condition_type String The type of check (e.g., 'variable', 'role', 'channel').
comparison String Operator to use ('equals', 'contains', 'starts_with').
value Any The value to compare against.

Manual JSON Override

For advanced users, you can inspect the raw JSON of any verb node. Here is the structure for a logic block checking if a user has "Admin" permissions.

logic_node.json 
{
  "id": "node_1284",
  "type": "verb_logic_if",
  "config": {
    "target": "user.roles",
    "operator": "includes",
    "value": "Admin"
  },
  "next_true": "node_1285",
  "next_false": "node_1290"
}
lightbulb

Pro Tip: Nested Logic

You can chain logic verbs to create "Else If" scenarios. However, for readability, we recommend keeping nesting to a maximum of 3 levels deep. Use the "Jump" verb to break out of deep nests.
